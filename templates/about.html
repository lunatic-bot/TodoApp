About Todo App

Your personal task manager, helping you stay organized and productive.

Our Mission

At Todo App, our mission is to provide a simple, intuitive platform that helps you manage your daily tasks efficiently. We believe that staying organized should be easy and accessible for everyone, and our app is designed with that principle in mind.

Features

  • Create, edit, and delete tasks with ease.
  • Mark tasks as completed to track your progress.
  • Organize tasks with priorities and due dates.
  • Clean and user-friendly interface for seamless task management.

Technology Stack

Our app is built using modern web technologies to ensure fast performance and a smooth user experience:

  • FastAPI: A high-performance web framework for building APIs with Python 3.7+ based on standard Python type hints.
  • Jinja2: A modern and designer-friendly templating engine for Python, used for rendering dynamic web pages.
  • Bootstrap: A responsive front-end framework for developing mobile-first websites, providing the app with a sleek and modern design.
